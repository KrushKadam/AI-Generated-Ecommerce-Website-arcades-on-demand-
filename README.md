# E-Commerce Website

## Project Overview
This project is a fully functional E-Commerce website developed as part of the ShadowFox Internship (Intermediate Level). The objective was to build a responsive and interactive online shopping platform using front-end web technologies.

## Features
- Product listing page
- Add to cart functionality
- Cart quantity update
- Product filtering
- Responsive design
- Clean UI/UX layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Learning Outcomes
- Improved DOM manipulation skills
- Understood event handling
- Strengthened responsive design concepts
- Learned real-world project structuring

## Challenges Faced
- Managing cart state dynamically
- Handling product filtering logic
- Debugging UI responsiveness issues
